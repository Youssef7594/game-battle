This game is a turn-based combat game where two characters face off. Each player takes a turn choosing an action (attack, defend, etc.)
to try and defeat their opponent. The first player to lose all their health points is defeated.

Technologies used :
This game was entirely developed using the Angular framework.
The game logic, turn management, and damage were handled by TypeScript.
HTML was used to build the interface and combat elements.
The game's design and visual atmosphere were created using CSS.
